# Terraform Infrastructure

This directory contains all Terraform infrastructure code for the project.

## Structure

Organize your Terraform code by:
- Environment (dev, staging, prod)
- Resource type (compute, networking, storage)
- Service or application

## Getting Started

1. Create your Terraform files (.tf)
2. Use the `just` commands from the root directory to manage infrastructure
3. Follow the project's naming conventions and best practices
